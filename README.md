# FlyCloudOne

Tu nube personal (Flask) con categorías y validación por tipo de archivo.

## Ejecutar
1) (Opcional) Crear entorno virtual
2) Instalar dependencias:
   pip install -r requisitos.txt
3) Iniciar servidor:
   aplicación Python.py
4) Abrir en el navegador:
   http://localhost:5000

## Estructura
- aplicación.py
- requisitos.txt
- manifiesto.json
- servicios.yaml
- plantillas/
  - índice.html
- estática/
  - logotipo.png
- subidas/
  - imágenes/
  - música/
  - documentos/
  - contactos/
  - correos/
  - vídeos/
