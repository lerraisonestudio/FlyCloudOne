# FlyCloudOne

Tu nube personal (Flask) con categorías y validación por tipo de archivo.

## Ejecutar
1) (Opcional) Crear entorno virtual
2) Instalar dependencias:
   pip install -r requirements.txt
3) Iniciar servidor:
   python app.py
4) Abrir en el navegador:
   http://localhost:5000

## Estructura

- requirements.txt
- manifest.json
- services.yaml
- templates/

- static/

- uploads/
